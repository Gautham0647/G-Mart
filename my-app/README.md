# Live project

https://aromacart.netlify.app/

# How to install and run locally?

`git clone : https://github.com/Gautham0647/G-Mart`
`cd Aroma Cart`
`npm install`
`npm start`

- Home Page
- Product Listing Page
- Single Product Page
- Filters by
